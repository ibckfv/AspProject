# RentalService ASP.NET Core (описание сгенерировано)

RentalService — это учебный проект сервиса аренды квартир.  
Приложение предоставляет полный цикл работы пользователя: регистрация и вход, просмотр списка объявлений, детальная страница квартиры, создание своих объявлений и бронирование.

## Основные возможности

- Аутентификация через JWT токены
- Управление объявлениями об аренде
- Бронирование квартир
- Управление пользователями
- PostgreSQL база данных

## Структура проекта

- `Controllers/` - Контроллеры для API и Views
- `Models/` - Модели данных
- `Services/` - Сервисы (JWT, хеширование паролей, и т.д.)
- `Data/` - DbContext для работы с базой данных
- `Views/` - Razor Views для отображения страниц
- `wwwroot/` - Статические файлы (CSS, JS)

## API Endpoints

### Аутентификация
- `POST /api/auth/register` - Регистрация пользователя
- `POST /api/auth/login` - Вход в систему
- `GET /api/auth/me` - Получить информацию о текущем пользователе

### Объявления
- `GET /api/apartments` - Список всех объявлений
- `GET /api/apartments/{id}` - Получить объявление по ID
- `GET /api/apartments/my` - Мои объявления (требует авторизации)
- `POST /api/apartments` - Создать объявление (требует авторизации)

### Бронирования
- `POST /api/bookings` - Создать бронирование (требует авторизации)
- `GET /api/bookings` - Список всех бронирований (требует авторизации)
- `GET /api/bookings/my` - Мои бронирования (требует авторизации)
- `DELETE /api/bookings/{id}` - Отменить бронирование (требует авторизации)
